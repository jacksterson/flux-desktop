## Bio / About Section

Flux is an open-source desktop widget engine for Windows, macOS, and Linux. Widgets are built with HTML, CSS, and JavaScript — no proprietary scripting, no platform lock-in.

I'm building this solo. Your support goes directly toward development time, keeping it free for personal use, and shipping the first official widget pack.
